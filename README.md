## 🧭 How I work

- **Human-in-the-loop by design** — confidence thresholds and review queues, because "the model is usually right" isn't a governance story.
- **Baselines before models** — if the LLM can't beat the regex, ship the regex.
- **Evaluation before enthusiasm** — every AI feature gets a labeled set and a threshold sweep before it gets a launch date.
